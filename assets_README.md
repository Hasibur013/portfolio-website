# Assets Folder

Place your files in this folder:

1. **Resume PDF**: 
   - Name: Hasibur_Resume.pdf
   - This will be linked in the "Download Resume" button

2. **Certificate PDFs** (optional):
   - Add your certification PDFs here
   - Update the links in index.html accordingly

3. **Blog Images** (optional):
   - Add blog post images
   - Reference them in blog.html

## Folder Structure:
```
assets/
├── Hasibur_Resume.pdf
├── certificates/
│   ├── certificate1.pdf
│   ├── certificate2.pdf
│   └── ...
└── blog-images/
    ├── post1.jpg
    ├── post2.jpg
    └── ...
```

## How to Add Your Resume:
1. Convert your current resume to PDF
2. Name it: Hasibur_Resume.pdf
3. Place it in the assets/ folder
4. The download button will automatically work

## Note:
Make sure to commit the assets folder to your GitHub repository so it's deployed with your site!
